# hu_portfolio

Expo + React Native + TypeScript로 만든 포트폴리오 앱입니다.

## 시작하기

의존성을 설치합니다.

```bash
npm install
```

개발 서버를 실행합니다.

```bash
npm start
```

실행 후 터미널의 안내에 따라 다음을 사용할 수 있습니다.

- `i` — iOS 시뮬레이터
- `a` — Android 에뮬레이터
- `w` — 웹 브라우저
- Expo Go 앱으로 QR 코드 스캔 (실기기)

## 스크립트

| 명령 | 설명 |
| --- | --- |
| `npm start` | Expo 개발 서버 실행 |
| `npm run android` | Android 에서 실행 |
| `npm run ios` | iOS 에서 실행 |
| `npm run web` | 웹에서 실행 |

## 기술 스택

- [Expo](https://expo.dev)
- [React Native](https://reactnative.dev)
- [TypeScript](https://www.typescriptlang.org)
